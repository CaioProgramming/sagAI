## 🎧 DJ Mixer Strategy

1. **Slider Balance**: The percentage next to each card represents where you should aim to place the
   slider in the interface (e.g., 40% = just under half-way for that specific instrument's
   prominence).
2. **Sync**: Ensure the **BPM** is set before starting the playback for best results.
3. **Tone**: Use the suggested keys to keep the composition harmonically consistent with the
   genre's "soul."
